# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

这是 CCCpc0 的个人 AI 提示词（Prompt Engineering）学习笔记仓库。使用者是不懂代码的初学者，本仓库仅包含 Markdown 笔记文件，没有代码、构建系统或测试。

## 仓库结构

- `AI提示词学习笔记.md` — 笔记正文，按序号章节组织，每条笔记包含"使用场景"和"原文"
- `GitHub同步指南.md` — 写给初学者的 Git/GitHub 同步教程
- `README.md` — 项目简介

## 笔记格式规范

新增笔记时，沿用现有格式：

```markdown
### N. 标题

**使用场景：** 一句话说明什么时候用

**原文：**

> 提示词内容（可多行）
```

## 语言

所有内容使用简体中文撰写。
